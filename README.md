# ASVR_HarKanJel_MRE
A collection of MRE (Mixed Reality Extensions) to have fun in AltSpaceVR

Once repository is cloned.
npm install
npm run build
npm run start


On every change to the code
git pull origin master





To stop the service
First look for the process ID
ps

Then kill the process
kill <processID>
or
Kill -9 <processID>
